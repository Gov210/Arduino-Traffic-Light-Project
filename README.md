This project demonstrates a simple Arduino based traffic light system that simulates the operation of a real world traffic signal. Using LEDs and an Arduino microcontroller, the system cycles through the standard traffic light sequence: Red → Green → Yellow → Red with programmable timing.

The project is designed for beginners who want to learn the fundamentals of Arduino programming, digital output control, and timing using the delay() function. It can also serve as a foundation for more advanced traffic management systems, such as pedestrian crossings, vehicle sensors, or smart traffic control.

Features:

Simulates a standard traffic light sequence
Controls Red, Yellow, and Green LEDs
Adjustable timing for each light
Simple hardware setup with Arduino
Beginner-friendly and easy to customize
Components Required:

Arduino Uno (or compatible board)
3 LEDs (Red, Yellow, Green)
3 × 220Ω resistors
Breadboard
Jumper wires
USB cable
